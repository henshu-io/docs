Henshu works in mobile browsers. The features are the same, but some interactions are different since there's no drag-and-drop on touch devices.

## Adding files to your episode

On desktop, you drag files from the Files tab into audio blocks. On mobile, there are two ways:

**Option 1: Place from the Files panel**
1. Tap to select one or more files in the **Files** panel
2. Tap **Place File(s) in Episode**
3. Tap where you want to place them. You'll see targets like "Add as new track" (inside an existing block) or "Add as new block" (at the end of a section)

**Option 2: Add Audio button**
1. Tap the **Add Audio** button within a section
2. Select the files you want from the list
3. Tap **Create Audio Block**

## Editor layout

The mobile editor is a single-column view. The Files and Properties panels are accessed through floating buttons (FABs) in the bottom-right corner. Tap any block or section to select it and the Properties FAB appears.

Playback controls and the scrubber sit at the bottom of the screen. Each section also has **Add Audio** and **Add Pause** buttons inline.

## Everything else

Editing audio (cuts, mutes, trimming), AI enhancement toggles, background music settings, generating, show notes, and account settings all work the same way on mobile. The interface adapts to the smaller screen but the functionality is identical.
